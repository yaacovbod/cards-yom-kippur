# היה או לא היה - מלחמת יום הכיפורים

## מה זה
משחק חידון לכיתה בסגנון "היה או לא היה". תלמידים מנחשים אם טענות היסטוריות על מלחמת יום הכיפורים (1973) הן אמיתיות או בדויות.

## הרצה מקומית
הפקודה להתקנת תלויות:
```
npm install
```
הפקודה להרצת שרת פיתוח:
```
npm run dev
```

## הוספת שאלות
קובץ השאלות נמצא בנתיב `src/data/questions.ts`. הוסף אלמנטים למערך לפי הפורמט:
```ts
{
  id: 2,
  imageSrc: '/stamps/image-name.jpg',
  text: 'הטענה ההיסטורית',
  isTrue: true,
  explanation: 'ההסבר שיוצג אחרי הניחוש',
}
```
תמונות שמור בתיקייה `public/stamps/`.

## טכנולוגיה
- הפרויקט: React 18 + TypeScript + Vite
- ספריית הקונפטי: canvas-confetti
- פונט: Heebo (Google Fonts)
